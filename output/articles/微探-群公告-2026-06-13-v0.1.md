# 微探群公告（v0.2.0）

感谢每一位付费支持微探的朋友！

微探是公众号内容的采集与洞察工作台：扫码即连、批量采集、本地洞察，一个 app 全搞定。

## 核心能力

- **采集引擎**：公众号搜索 / 采集 / 续抓，多模式续抓、随时可打断，一键补齐缺失正文
- **三栏工作台**：账号列表 / 文章流 / 正文阅读（HTML 与 Markdown 双视图）
- **账号洞察**：基本信息、发文趋势分析、文风分析
- **GitHub 归档**：OAuth 一键接入，文章自动同步到你自己的仓库，数据永远在你手里
- **开箱即用**：内置采集内核（无需装 Python），app 内扫码登录（不依赖 Chrome），自动增量更新
- **全平台**：macOS（Intel / Apple Silicon）、Windows、Linux

## 最近更新（v0.2.0）

- GitHub OAuth 接入，文章自动归档到自己的仓库
- 多模式续抓、抓取可中断、本地文件一键定位
- 一键补齐所有缺失正文
- 登录失效一键重连，状态自动同步
